Police iAPS

iAPS is a web based application which can be used in prisons to manage the daily activities of a prison. This project is made on Django and MySQL. Frontend technologies like Bootstrap has also been used.

Pros & Cons
Current System

In the existing system, all the works are done by manually. The registration of jailers, allocating tasks to prisoners, giving their salaries, etc are stored in traditional book keeping system. The main limitation of the system is that it is time consuming process.

Pros & Cons
Proposed System

The purpose of the project is to describe the functionality and specifications of the design of a web application for managing jailers, health officers, prisoners and their payroll. Now with the help of this system the admin has the information on his finger tips and can easily prepare a good record based on their requirements.

Offered services

Correctional Homes are necessary for the well-being of a soceity. iAPS systems like this can help jailors and health officers with the following services

PAYROLL

We can efficiently manage the payroll of inmates for the works they do inside the prison.

PRISONER DATA

In this era, data is of atmost importance. With iAPS, we safely store prisoner data.

PAROLES

iAPS can make process of granting paroles quicker at the administrative level.
